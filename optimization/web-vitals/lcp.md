# LCP

The **render time** of the **largest content element** visible **in the viewport**.

### Content elements:

* Foreground images
  * `<img>`
  * `<image>` into the `<svg>`
* Contentful background images
  * The element referencing the background image
* Video elements
  * with poster images
* Text
  * block-level elements containing text
